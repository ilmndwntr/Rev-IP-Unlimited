# Rev-IP-Unlimited
Rev-IP Unlimited

## Note 
Pakai BASH untuk menjalankanya, tools ini mengunakan bahasa python3 jadi anda harus install python3 terlebih dahulu

## How to use
* $ git clone https://github.com/ilmndwntr/Rev-IP-Unlimited.git
* $ cd Rev-IP-Unlimited
* $ chmod +x RevPy
* $ ./RevPy
